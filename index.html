Even Columns

Define a grid class container. Add a cols-# class to define the number of columns, ex. cols-3 yields 3 even columns. gap-# defines grid gap, up to 50 in increments of 5. 15 columns is maximum.

Uneven columns

Inside cols-#, each item gets a col-span-# class to show the columns it spans, ex. col-span-5 in cols-7 will span 5/7 columns.
